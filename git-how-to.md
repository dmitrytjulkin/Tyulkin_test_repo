# Как создать ssh ключ
Чтобы создать ssh ключ в терминале находим местоположение папки .ssh. Далее прописываем команду (в кавычках прописываем свою почту):

'''
ssh-keygen -t ed25519 -C "your@email"
'''

Терминал будет спрашивать нас о названии файла, в котором будет храниться ключ, и пароль для него. В конце, в папке .ssh появятся два файла, один из них - текстовый.

# Как добавить ключ в аккаунт на GitHub
В этом текстовом файле находим строку, которую надо скопировать.

Открываем Github. Заходим в настройки профиля. Находим раздел "SSH and GPG keys". Там добавляем новый ssh-ключ, посредством вставления скопированной строки.
